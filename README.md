# 🔐 Django Authentication with OAuth 2.0
## Projeto desenvolvido durante o curso "Django: autenticação com OAuth2.0" da Alura, como parte da trilha de Desenvolvimento Backend com Python.

# 📚 Sobre o Projeto
## Aplicação Django com autenticação via OAuth 2.0, permitindo que usuários façam login utilizando provedores terceiros — como o GitHub — sem precisar criar uma conta local na plataforma.

# ✅ O que foi aprendido e implementado

- Funcionamento do protocolo OAuth 2.0 de forma prática
- Autenticação com provedores externos (GitHub)
- Proteção de rotas para apenas usuários autenticados
- Integração do painel Admin do Django com OAuth
- Extensão do conhecimento para outros provedores OAuth
